Arrays工具类

2020.06.23 山水 Java
  1. Arrays 工具类可以对数组进行二分查找，支持基本的数据类型和范型，对于范型需要传入自定义的比较器。
    值得注意的是Java内部实现的时候对除二操作采用了移位运算。而且它的返回值不是返回-1而是返回数组low没有找到的位置
    static int binarySearch(double[] a, double key)
    static <T> int binarySearch(T[] a, int fromIndex, int toIndex, T key, Comparator<? super T> c)
  2. copyOf 方法即可对数组进行缩小也可以放大，缩小会截断，放大的话对于基本数据类型的数组会自动补全初始值，
    对于引用类型，自动补null。内部实现是创建一个新数组，然后使用System.arraycopy本地方法进行复制
    static float[] copyOf(float[] original, int newLength)
  3. Arrays.copyOfRange 内部实现和copyOf一样，只是可以指定从那个位置开始。
  4. Arrays.equals 方法可以比较两个数组的内容是否相等，但是如果元素相同但是顺序不同也不会相等
    内部使用for循环实现。
    static boolean equals(long[] a, long[] a2)
  5. Arrays.fill方法可以对一个数组进行填充，内部使用for循环实现。
    static void fill(boolean[] a, boolean val)

最后不断缩小间隔，使全部内容有序。
