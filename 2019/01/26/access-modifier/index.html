Java访问修饰符

2019.01.26 山水 Java

对于类，只有两种public default.

  1. public所有类都可以使用该类。
  2. default只有同一包下的类才可以使用该类。

对于成员(成员方法，成员变量)，有public，protected，default，private。

  1. public就不多说了
  2. protected,同一包类可以访问，不同包类其它类不能访问，子孙类也只可以访问其静态成员
  3. default 同一包下随便访问，不同包下不能访问。子类也不行。
  4. private，只有当前类可用。

访问修饰符与继承体系

对于类

父类为public随便继承，父类为default，那么只有在同一包中才能继承继承了。

对于成员

  1. public修饰的成员，子类都会继承。
  2. protected修饰的成员。无论是静态还是动态，都会继承。
  3. default成员。同一包下子类随便继承，不同包下，子类不能继承。
  4. private 成员，只属于父类私有，不会传递给下一级。

总结

对于类

  1. 对于类无论是继承，还是不继承。public可以访问，default同一包下可以访问。

对于成员

  1. public成员：既能被继承也能被访问，没有限制
  2. protected成员：
    同一包下：
    既能被继承也能被访问

    不同包下(Y/N的情况)：
    1.普通类不能访问也不能继承
    2.子类可以访问protect静态成员，继承所有protect成员
  3. default成员：同一包下可以被继承可以被访问，不同包下不能访问也不能继承。
  4. private成员：不管属于哪个包，都不能被继承，也不能被访问，只有本类可以用。
