CentOS上JDK，TOMCAT，MYSQL的安装

环境：centos7 ，jdk18，mysql5.7

JDK8的安装

1.安装基本工具

yum install wget
yum install vim

2.使用wget下载jdk.rpm包

64位
wget --no-cookie --no-check-certificate --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u191-b12/2787e4a523244c269598db4e85c51e0c/jdk-8u191-linux-x64.rpm -O jdk-8u102-linux-x64.rpm
如果是32系统，请下载32包

3.使用rpm命令安装jdk

rpm -ivh jdk8.rpm

4.检验是否安装成功

java-version
java
javac

tomcat的安装

1.使用wget下载tomcat.tar.tgz压缩包

wget http://archive.apache.org/dist/tomcat/tomcat-9/v9.0.8/bin/apache-tomcat-9.0.8.tar.gz

2.使用tar命令解压

tar -zxvf tomcat.tar.gz

3.复制tomcat到/usr/tomcat目录下

mv tomcat /usr/tomcat

4.进入tomcat目录

cd /usr/tomcat/

5.配置tomcat_home环境变量

vim /etc/profile
export TOMCAT_HOME=/user/tomcat/

6.进入tomcat的bin目录，启动tomcat

cd /usr/tomcat/bin
chmod +x *.sh # 赋予可执行权限
./startup.sh

7.更改tomcat的默认端口号为80

vim /usr/tomcat/conf/server.xml
找到<Connector port="8080" protocol="HTTP/1.1"
改为80

安装mysql

1.下载mysql软件源

wget https://dev.mysql.com/get/mysql80-community-release-el7-1.noarch.rpm

2.安装软件源

rpm -Uvh mysql.rpm

3.配置mysql安装文件

vim /etc/yum.repos.d/mysql-community.repo
把需要的安装数据库enable改为1，不需要的改为0

4.安装mysql

yum install mysql-community-server

5.查看mysql自动生成的初始密码

vim /var/log/mysqld.log
找到其中的temporary password

6.使用初始密码登录

mysql -uroot -p

7.降低mysql对密码的检验强度

set global validate_password_policy=0;
set global validate_password_mixed_case_count=0;
set global validate_password_number_count=3;
set global validate_password_special_char_count=0;
set global validate_password_length=3;

8.设置新密码

SET PASSWORD FOR 'root'@'localhost' = PASSWORD('123');
