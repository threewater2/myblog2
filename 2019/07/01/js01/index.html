JS变量声明&数据类型

2019.07.01 山水

一般我们推荐使用严格模式
变量必须先声明后使用。
声明变量的两种方式
let：赋值后可以被改变，驼峰式命名法
const：赋值后，不能被改变，大写，多个单词之间用_分割

let 和 var 的区别

var的作用域只有两种，全局的和函数的，也就说它会冲出if，for等代码块的限制
var的声明操作发生在，全局和函数开始之前，赋值发生在赋值语句处

和Java一样，只是引用。

数据类型

  1. number类型：包括，整形，浮点，Infinity, -Infinity ，NaN
    1/0 是无穷，计算错误或者未定义的数学运算返回NaN,你可以大胆的做数学运算
    NaN不等于任何数
  2. string类型
  3. boolean类型
  4. null类型，null不是空指针，或者空引用，他就是字面意思，表示我也不知道这是什么
  5. undefined类型，这个就表示空指针或者空引用了，一般我们用来检查变量是否被分配。
  6. symbol 是object的UUID
  7. object 类型，function属于object类型，只不过typeof把它视为了function

关于小数

小数是分数的一种表示形式。不如1/10，就是0.1。但是1/3是0.333…无限循环。
可见，如果存储容量有限我们是无法存储1/3这样的小数的，我们只能近似的存储它：0.333334。
这个问题在二进制中同样存在，表示的数总会大一点或者小一点，
这样如果两个不精确的数做运算，其结果必然是不精确的，这样他就不会等于理想结果

我们只能用0.123213..toFixed(3)这样精确到多少位的字符串表示
但是有些小数参加运算有问题，有些又不会，

String到Number的严格转换和松转换

  1. 严格转换：+，Number()
  2. 松转换：parseInt(),parseFloat(),遇到不是数字就不会往后面读取了，如果第一个也不是数字，返回NaN
