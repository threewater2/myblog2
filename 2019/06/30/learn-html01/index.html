#图片像左无限延申

如果设置了margin或者border那么height或者width要减

测量的话把border也测进去，先把布局画出来，后期再减去

#字体大小
font-weight：500
不能带单位
字体有默认的上下空白
所以我们在设置边距的时候应该少设一点

定位

一般我们可以用marigin 或者padding实现
但是如果mairign很长，那么就不适合，
还有比如如果不知道设置多少，可以以用position

如果父元素没有设置定位，那么子元素相对于浏览器进行定位

相对定位没啥作用，一般是和绝对定位使用
marigin是在同一层的，而position是在不同层的
定位的层次是按html标签排列的顺序决定的，越后面的元素，越在上层

透明

opacity
rgba

transition

适用于所有连续变化的属性，不连续变化的没用
也就说这个属性设在那个标签上，如果这个标签有瞬间变化，他就会把他转成连续变化

#overflows:hidden
只要父，祖父元素设置了这个，任何超出的部分都会被隐藏

#hover属性的深度理解
父元素的范围会包括子元素

Block

  1. 有margin 和 padding 四边都有。
  2. 独占一行

Inline

  1. width，height无效，由内容大小决定。
  2. margin padding 左右有效，上下无效

Inline Block

例如 img input标签
属于Inline元素下

两个Inline Block 标签在一起时 中间会有缝隙
第一个元素加float：left

CSS样式会影响标签的元素类型

元素类型转换

这个是有需求的，因为标签代表功能，如果这个标签不能满足CSS需求的话
就要进行转换
例如：a inline—->inline block

#图片水平垂直居中
父元素 设置
text-algin=center；
line-height：同样高度。按理说这样就可以水平垂直居中了，但是不够精确
子元素 设置
vertical-align：middle；使垂直居中更精确，只对iline-block有用
