线程之间的协作工具类

2019.03.12 山水 Java

wait() /notify() 是最基本的，Java为我们提供了一些线程协作的工具类。

ReentrantReadWriteLock

读写锁，他有两个锁读锁和写锁，和一个等待队列，如果一个线程持有读锁，它可以唤醒队列中前面想要读的线程。所以它的读可以说是并行的。如果一个线程持有写锁，那么任何线程都不能进入。

Semaphore

信号量，用来限制并发访问数量。

ThreadLocal
