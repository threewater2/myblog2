命令设计模式

2019.03.06 山水 Java
  1. ICommand： 接口 命令这一名词的抽象，所有具体的命令都必须继承它
  2. 实体：命令接收者，负责接收命令，并执行具体操作
  3. 发送者：或者说具体命令。负责发送某个具体的命令给实体，因此它应该实现ICommand接口
  4. 管理器：负责管理所有的命令，支持添加多个命令。
